Simple Smart Voting System
📌 Project Description

The Simple Smart Voting System is a blockchain-based smart contract that enables transparent, secure, and tamper-proof voting. Built using Soroban Smart Contracts on the Stellar blockchain, this project demonstrates a minimal yet functional decentralized voting mechanism.

This project is ideal for beginners who want to learn how smart contracts can be used to build real-world governance and decision-making systems.

⚙️ What It Does

Allows users to cast a vote for a predefined option

Ensures each voter can vote only once

Stores vote counts immutably on the blockchain

Provides real-time vote results from the smart contract

✨ Features

🔐 Transparent & Trustless – Votes are stored on-chain and cannot be altered

🚫 One Vote Per Address – Prevents double voting

⚡ Fast & Low Cost – Powered by Stellar’s Soroban smart contracts

🧩 Simple & Beginner-Friendly – Easy-to-read contract logic

🌍 Decentralized – No central authority controlling the votes

🛠️ Tech Stack

Blockchain: Stellar

Smart Contract Platform: Soroban

Language: Rust

Network: Stellar Testnet

🚀 How It Works

Deploy the smart contract to the Stellar network

Initialize the voting options

Users submit a vote transaction

The contract verifies voter eligibility

Vote count is updated and stored on-chain
